一、四大盈利模式

  1. 信息差
    我知道的，你不知道
  2. 认知差
    我懂得的，你不懂
  3. 执行差
    你我都懂，你不做我做
  4. 竞争差
    你我都做，我做得比你好

核心决策维度：

  • 卖服务还是卖产品？
  • 卖多贵？卖多久？卖几次？

二、火箭定律与第一性原理

“火箭发射前期消耗六分之一燃料，进入轨道后平稳运行。事物前期准备最耗精力。”
一个粗糙的开始就是最好的开始，先做个【MVP】再逐步【迭代】。

马斯克第一性原理框架

  1. 看到事物本质问题
  2. 拆解问题
  3. 找到最佳解决方案
  4. 质疑一切（除物理定律），打破固有思维

关键应用领域

  • 产品设计：白痴指数 = 成品价/材料成本（越低越好）
  • 职场发展：核心是创造稀缺性价值，而非苦劳
  • 创业方向：立足市场需求，非创始人喜好
  • 健康管理：建立良好习惯>事后保险
  • 教育本质：培养探索能力>追求分数

三、商业本质论

赚钱逻辑

  1. 价值交换：打造受欢迎的产品
  2. 销售闭环：
    • 找到首批种子用户
    • 构建口碑裂变机制

《教父》启示：
“看透本质者与混沌者，人生截然不同”

四、互联网创业11步法

需求验证阶段

  1. 洞察需求
  2. 打磨需求
  3. 总结需求文档
  4. 制作产品原型图

产品开发阶段

  1. 构建核心功能
  2. 打造MVP（最小可行产品）
  3. 验证MVP有效性

市场扩张阶段

  1. 获取1000个种子用户
  2. 设计变现路径
  3. 验证商业模式
  4. 规模化市场投放

五、EBSI象限

E型：普通雇员
B型：利用一个系统为你赚钱
S型：生意人，提供产品和服务，工作越努力越无法抽身
I型：投资性，钱生钱，投资B
左侧ES:追求安全， 右侧BI:追求自由

六、30万存款定律

当你的存款突破30万，整个世界的镰刀都在嗡嗡作响……

有人让你分期买50万的豪车；有人哄你回城买空置的房产；
有人让你加盟各种店铺；
大家都在热心“帮助”你。
如果你信了，那这30万就不是起点，而是万丈深渊。那些教你钱生钱的教程背后,隐藏着无数的返贫陷阱。那些承诺阶层跨越的豪车展厅,停满了中产的坟墓。
